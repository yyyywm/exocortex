# 依赖与工具

## 1. 必需软件

| 软件 | 最低版本 | 用途 | 安装方式 | 验证命令 |
|------|----------|------|----------|----------|
| Git | 2.40 | 版本控制 | [git-scm.com](https://git-scm.com/) | `git --version` |
| Node.js | 20.x | Quartz 构建 | [nodejs.org](https://nodejs.org/) 或 nvm | `node --version` |
| npm | 10.x | 包管理 | 随 Node.js 安装 | `npm --version` |
| Obsidian | 最新 | 笔记编辑 | [obsidian.md](https://obsidian.md/) | 图形界面 |
| Claude Code | 最新 | AI 执行器 | `npm install -g @anthropic-ai/claude-code` | `claude --version` |

### Git 配置（必需）

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
git config --global init.defaultBranch main
```

### GitHub SSH Key（推荐）

```bash
ssh-keygen -t ed25519 -C "your.email@example.com"
cat ~/.ssh/id_ed25519.pub
# 将公钥添加到 GitHub Settings → SSH and GPG keys
```

## 2. Obsidian 插件清单

在 Obsidian 中安装以下社区插件（Community Plugins）：

| 插件名 | 用途 | 配置要点 |
|--------|------|----------|
| **Obsidian Git** | Git 自动备份与同步 | 启用自动备份，设置备份间隔 5 分钟 |
| **Templater** | 模板引擎 | 模板文件夹设为 `templates/` |
| **Dataview** | 笔记查询与列表 | 无特殊配置，按需使用 |
| **Excalidraw** | 手绘图表与示意图 | 可选，用于画系统架构图 |
| **Copilot** | AI 问答（Obsidian 内） | 配置 API Base URL（Claude/Hermes） |

### 插件安装步骤

1. Obsidian → Settings → Community Plugins → Turn on community plugins
2. Browse → 搜索插件名 → Install → Enable

## 3. Quartz 安装

Quartz 不是全局安装的，而是作为项目依赖：

```bash
# 在项目根目录执行
npx quartz create

# 或克隆官方仓库
git clone https://github.com/jackyzha0/quartz.git
cd quartz
npm install
```

## 6. 版本锁定

为了确保可复现性，以下版本组合经过验证：

```
Git: 2.40+
Node.js: 20.10+ LTS
npm: 10.2+
Quartz: 4.2+
Obsidian: 1.5+
Claude Code: 0.32+
```

如遇兼容性问题，优先检查 Node.js 版本。

## 7. 环境变量（可选）

如需在 Claude Code 中使用 API Key，可配置：

```bash
# ~/.bashrc 或 ~/.zshrc
export ANTHROPIC_API_KEY="sk-ant-..."
export OPENAI_API_KEY="sk-..."  # 如使用 Claude Code 的 OpenAI 兼容模式
```

## 8. 快速检查脚本

创建 `check-env.sh` 用于快速验证环境：

```bash
#!/bin/bash
echo "=== Digital Garden 环境检查 ==="
echo "Git: $(git --version 2>/dev/null || echo '未安装')"
echo "Node: $(node --version 2>/dev/null || echo '未安装')"
echo "npm: $(npm --version 2>/dev/null || echo '未安装')"
echo "Claude Code: $(claude --version 2>/dev/null || echo '未安装')"
echo "Obsidian: $(obsidian --version 2>/dev/null || echo '请手动检查')"
echo ""
echo "Git 用户: $(git config user.name 2>/dev/null || echo '未配置')"
echo "Git 邮箱: $(git config user.email 2>/dev/null || echo '未配置')"
```
